# boite-outils-secrete
Un mini-projet de chiffrement/déchiffrement de messages (Code César) en Python, réalisé en binôme pour s'initier aux méthodologies Scrum et au workflow Git/GitHub.
### Quelque commande pour la preparation et l'execution de l'environnement :

* Commande de **creation** d'environnement Python : 
    ```
    python3 -m venv .venv
    ```
* Commande d'**activation** de l'environnement Python :
    ```
    source .venv/bin/activate
    ```
* Commande de **desactivation** de l'environnement Python :
    ```
    deactivate
    ```
* Commande d'**execution** : 
    ```
    python3 <nom-du-fichier-a-executer>.py
    ```
